myapp 1.2.3

Description:
The animal command.

Usage:
    myapp animal [LEGS] [OPTIONS] <COMMAND>

Examples:
    myapp animal --help

Arguments:
    [LEGS]    The number of legs

Options:
    -h, --help     Prints help information
    -a, --alive    Indicates whether or not the animal is alive

Commands:
    dog <AGE>    The dog command
    horse        The horse command
